# VS Code Keyboard Shortcuts

## General Editing

- **Ctrl + C**: Copy the selected text
- **Ctrl + X**: Cut the selected text
- **Ctrl + V**: Paste
- **Ctrl + Z**: Undo
- **Ctrl + Y**: Redo
- **Ctrl + A**: Select all
- **Ctrl + S**: Save current file
- **Ctrl + Shift + S**: Save all files
- **Ctrl + F**: Find in current file
- **Ctrl + H**: Replace in current file
- **Ctrl + / **: Toggle line comment
- **Shift + Alt + A**: Toggle block comment

## Navigation

- **Ctrl + P**: Quick open file
- **Ctrl + G**: Go to line
- **Ctrl + T**: Show all symbols
- **Ctrl + Shift + O**: Go to symbol in file
- **Ctrl + -**: Navigate back
- **Ctrl + Shift + -**: Navigate forward
- **Ctrl + B**: Toggle sidebar visibility
- **Ctrl + `**: Toggle integrated terminal

## Multi-cursor and Selection

- **Alt + Click**: Insert cursor
- **Ctrl + Alt + Down / Up**: Add cursor below / above
- **Ctrl + D**: Add selection to next find match
- **Ctrl + L**: Select current line
- **Shift + Alt + Right / Left**: Expand / shrink selection

## Search and Replace

- **Ctrl + Shift + F**: Search across files
- **Ctrl + Shift + H**: Replace across files
- **F3** / **Shift + F3**: Find next / previous
- **Alt + Enter**: Select all matches

## Window and Panel Management

- **Ctrl + \**: Split editor
- **Ctrl + 1 / 2 / 3**: Focus first/second/third editor group
- **Ctrl + K Z**: Zen mode
- **Ctrl + Shift + E**: Show Explorer
- **Ctrl + Shift + F**: Show Search
- **Ctrl + Shift + G**: Show Source Control
- **Ctrl + Shift + D**: Show Run and Debug
- **Ctrl + Shift + X**: Show Extensions

## Debugging

- **F5**: Start debugging / continue
- **Shift + F5**: Stop debugging
- **F9**: Toggle breakpoint
- **F10**: Step over
- **F11**: Step into
- **Shift + F11**: Step out

## Terminal

- **Ctrl + Shift + `**: Create new terminal
- **Ctrl + C**: Copy selection in terminal
- **Ctrl + V**: Paste into terminal
- **Ctrl + Shift + C**: Copy path of active file

## Useful Tips

- **Ctrl + K Ctrl + S**: Keyboard shortcuts reference
- **Ctrl + Shift + P**: Open Command Palette
- **Ctrl + Space**: Trigger suggestion / IntelliSense
- **Ctrl + Shift + M**: Show problems panel
- **Ctrl + K Ctrl + C**: Add line comment
- **Ctrl + K Ctrl + U**: Remove line comment

> Note: These shortcuts are for Windows. On macOS, replace `Ctrl` with `Cmd` and `Alt` with `Option` where needed.
